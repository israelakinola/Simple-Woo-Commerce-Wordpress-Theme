# Simple Woo-Commerce Wordpress Theme
This is a free WordPress WooCommerce Theme for your simple store. The store is the home-page, and you can post and sell a variety of products. You will have to create any additional pages through WordPress Classic Editor, Gutenberg, or Elementor if you need one. This theme also offers responsive Header and Footer navigation menus for any other pages you created. Regular updates and development will be provided as time goes on. Don't forget to update when an update is available.

### REQUIREMENT
Woo Commcerce Plugin - This store theme requires the Woocommerce Plugin to work.

### INSTALATION
1. Install and Setup Woo Commerce.
2. Download this project as a Zip on the Github Repo https://github.com/israelakinola/Simple-Woo-Commerce-Wordpress-Theme
3. Extract zip and upload on your wp-theme directory, or through the Appearance Admin Page.
4. Setup ygour Header Menu and and add the pages pages, custom-links of all your top navigation menu
5. Setup your Help Menu and add the pages, custom-links of where your customer can get help and assistance
6. Setup your Social Menu and add the custom-links to your social media accounts.
7. Upload your Logo in the Wordpress Appearance Admin Page. Navigate to Customize/Site-Identity subpage.
