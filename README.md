# Simple Woo-Commerce Wordpress Theme
This is a free simple WordPress Woo Commerce Theme for your simple Store. The Store is the home page, and you can post and sell a variety of products. You will have to create any additional pages through WordPress Classic Editor, Gutenberg, or Elementor. This theme also offers responsive Header and Footer navigation menus for any other pages you created.

### REQUIREMENT
Woo Commcerce Plugin - This store theme requires the Woocommerce Plugin to work.

### INSTALATION
1. Install and Setup Woo Commerce.
2. Setup Your Head Menu and and add the pages pages, custom-links of all your top navigation menu
3. Setup Your Help Menu and add the pages, custom-links of where you customer can get help and assistance
4. Setup Your Social Menu and add the custom-links to your social media accounts.
5. Upload your Logo in the Wordpress Appearance Admin Page. Navigate to Customize/Site-Identity subpage.

### USAGE
Start Posting Products
